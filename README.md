# FSScene
###### a fallstruktur scene variable editor

## about
this is a pretty simple tool developed in order to aid the modification of scene
variables for
[FALLSTRUKTUR](https://store.steampowered.com/app/3674270/FALLSTRUKTUR/).

it works by finding the location of the original variables, and replacing them
with new ones. it provides a `sceneConfiguration.json` file which is used to
keep the configuration you use.
